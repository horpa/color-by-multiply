# How to Use Your Worksheet (English)

This guide is for **students**. It helps you use the paper your teacher gives you.

You will see a **picture made of colored squares** and a list of **math exercises** below it.

## What is on the paper?

Your worksheet has three parts:

1. **The big grid** — a picture with colored and white squares.
2. **The color key** — small colored circles with numbers.
3. **The exercises** — math problems to solve.

## Blue and green numbers

Look at the grid:

- **Blue numbers** on the **left** tell you the **row** (which row a square is in).
- **Green numbers** on the **top** tell you the **column** (which column a square is in).

```
        1   2   3   4   ...   (green = columns)
      ┌───┬───┬───┬───┐
  1   │   │   │ ■ │   │       (blue = rows)
  2   │   │ ■ │   │   │
  3   │ ■ │   │   │   │
      └───┴───┴───┴───┘
```

To find a colored square:

1. Look **across** to the green number on top → that is the **column**.
2. Look **down** to the blue number on the left → that is the **row**.

**White squares** are empty background. You do not need math problems for those.

## The color key

Below the grid you see colored circles with numbers (1, 2, 3…).

Each color on the picture has a number. The same colors appear next to the exercises so you know **which square** each problem belongs to.

## The exercises

Each exercise is a small math problem. You may see **×** (multiply) or **÷** (divide).

Some numbers are already written. One spot is an **empty box** where **you** write the answer:

| Box color | What to write |
|-----------|---------------|
| **Light blue box** with dark blue edge | Write the **row** number |
| **Light green box** with dark green edge | Write the **column** number |

Numbers written in **blue** are row numbers. Numbers written in **green** are column numbers.

### Examples

**Multiplication**

- `[blue box] × 4 = 12` → write the **row** number in the blue box.
- `3 × [green box] = 12` → write the **column** number in the green box.

**Division**

- `12 ÷ [blue box] = 4` → write the **row** number in the blue box.
- `12 ÷ 3 = [green box]` → write the **column** number in the green box.

## How to solve one exercise — step by step

Try this:

1. Look at the **small colored circle** at the end of the exercise. Remember its color and number.
2. Find a square on the big grid that has **the same color**.
3. Read the **blue row number** and the **green column number** for that square.
4. Look at the exercise. Is the empty box **blue** or **green**?
   - **Blue box** → write the **row** number.
   - **Green box** → write the **column** number.
5. Write your number **neatly inside the box**.

### Example

Exercise: `3 × [green box] = 12` with a red color chip.

- The red square is in **row 3** and **column 4**.
- The empty box is **green**, so you write the **column** number: **4**.

Check: 3 × 4 = 12. Correct!

## Tips

- Write one digit clearly inside each box.
- Use the color chip to find the right square on the grid.
- Blue always means **row**, green always means **column**.
- Take your time — read the row and column carefully.
